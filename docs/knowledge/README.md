# AGENT OFFICE 지식 패키지

AGENT OFFICE 에이전트들이 사용하는 전문 지식 문서 저장소.
**원문 수정 금지** — 검증된 카피 라임·법적 기준이므로 임의 요약·변형 불가. (학습으로 인한 지침 추가는 대표 승인 후 "📈 학습으로 추가된 지침" 섹션으로만 추가 가능)

## 폴더 구조

```
docs/knowledge/
├─ legal/
│   └─ 노무자문_페르소나.md          [문서 A] → 지율 (법무팀장)
├─ marketing/
│   ├─ 문자톡톡_전문가_지침.md       [문서 B] → 글샘·한결
│   ├─ 상품별_톤앤매너_가이드.md      [문서 C] → 글샘·한결
│   ├─ 마케팅_문자_가이드북.md        [문서 D] → 글샘·한결
│   ├─ 명분_라이브러리.md            [문서 E] → 글샘·한결
│   ├─ 검증된_카피_자산집.md         [문서 F] → 글샘·한결
│   └─ 마케팅_전문팀_시스템.md        [문서 G] → 한결·미소·글샘
└─ company/
    ├─ 비전과목표.md                [문서 H] → 마루·한결·예리·기안
    └─ 운영_지시규칙.md              [문서 I] → 마루·세미
```

## 에이전트 ↔ 지식 매핑

에이전트별 매핑은 DB `agents` 테이블의 `knowledge_files` 필드에 저장됨 (AI 호출 연결은 후속 차수).

| 에이전트 | 역할 | 지식 문서 |
|---|---|---|
| 마루 (실장) | 오더 접수·팀 배정·통합 보고 | I + H |
| 한결 (마케팅팀장) | 지시 정리·검수 | B~G + H |
| 미소 (디자인) | 이미지/영상 프롬프트 | G |
| 글샘 (문구) | 카피라이팅 | B~G |
| 예리 (분석) | 인스타 성과·경쟁 분석 | H |
| 세미 (회계조회) | 정산·매출 조회 | I |
| 지율 (법무팀장) | 노무 자문 보조 | A |
| 기안 (기획) | 기획 보고서 | H |

## 표시명 치환 규칙 (원문은 수정 금지, 화면 출력 시에만 치환)

- "팀장 똑똑이" → **한결** / "카피라이터 글잘이" → **글샘** / "비주얼 디렉터 그림이" → **미소** / "노무법인 똑똑이 대표 노무사" → **지율**
