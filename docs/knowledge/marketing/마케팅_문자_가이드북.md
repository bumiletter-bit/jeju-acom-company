## 3장. 명분 우선 원칙 (최우선!)
범 대표님 직접 정립: '명분 없는 할인은 절대 금지!'

명분이 있어야 하는 이유:
- 명분 없는 할인 = 싸구려 느낌, 브랜드 가치 훼손, 신뢰도 하락
- 명분 있는 할인 = 특별함, 감사함, 단골 우대감 → 재구매 강화
- 황금구조 5단계 중 ③ 명분 제시가 빠지면 문자 전체가 무너짐

### 5가지 명분 카테고리
① 회사 성장·이정표 명분: 농업회사법인 새출발 감사 / 자사몰 akkome.com 오픈 기념 / 체험장 3,000명 돌파 감사 / 5,000평 귤밭 매입 기념 / 선과장 오픈 기념 (2027)
② 계절·제철 명분: 지금이 1년 중 가장 맛있을 때 / 봄 제철 한정·겨울 수확 직송 / 이 시기 놓치면 내년을 기다려야 합니다 / 첫 수확분 한정 입고 / 초반 호박이 1년 중 가장 포슬
③ 고객 관계 명분: VIP 단골 고객님께만 먼저 / 재구매 고객 감사 / 1년 만에 돌아온 제철 인사 / 단골 고객 전용 사전 안내
④ 상품 특성 명분: 한정 수량 입고·농장 직접 수확분만 / 검증된 품질만 / 신상품 첫 출시 기념 / 왁스코팅 0% (레몬) / 후숙 불필요 (하우스한라봉)
⑤ 행사·시즌 명분: 명절 (설·추석) 선물 시즌 / 어버이날·어린이날·스승의 날 / 연말 감사 이벤트 / 체험장 시즌 오픈·마감 / 가정의 달 5월

## 5장. 채널별 규격 (알리고 vs 톡톡)

### 알리고 SMS/LMS 규격
- SMS: 90자 이내 (짧은 푸시·마감 임박용)
- LMS: 1,000자 이내 (메인 안내·VIP 발송)
- LMS 제목란: 40byte 제한 (한글 약 13자)
- 이모지: ❌ 전부 ?로 깨짐
- 특수기호: ✅ ★ ▶ ◆ ━ ─ 사용
- 첫 줄 (광고) 표기: ✅ 필수 (법적 의무)
- VIP 혜택 블록: ✅ 하단 고정 삽입
- 수신거부 안내: ✅ 필수

### 네이버 톡톡 규격
일반 메시지: 본문 약 400자 이내 (클릭률 ↑), 이모지 ✅ 사용 가능 (🍊🎁⏰💥💛)
이미지형 카드: 제목 1줄 (한글 약 25~30자) / 내용 최대 8줄 (총 약 200~240자) / 버튼 최대 2개 (각 14자 이내) / 이미지 권장 628x380px, 비율 1:1까지 허용
