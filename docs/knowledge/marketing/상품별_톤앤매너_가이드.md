감귤·만감류·옥수수·채소·나물·향후 라인업까지
제주아꼼이네 농업회사법인(주) — 2026년 6월 작성

## 이 가이드 사용법
- 각 상품마다 '시즌 / 킬러 셀링 포인트 / 권장 톤 / 검증 카피' 4가지 정리
- 문자·톡톡 카피 작성 시 해당 상품 페이지를 먼저 참고
- 킬러 셀링 포인트는 모든 카피에 반드시 1회 이상 노출
- 검증된 카피 샘플은 표현 그대로 활용하거나 변형 가능

## Part 1. 감귤·만감류 (메인 매출)

### 🍊 노지한라봉
- 📅 시즌: 12월~3월 (겨울 제철)
- 💎 킬러 셀링 포인트: 노지에서 자연 그대로 자란 깊은 단맛 / 한겨울 햇살의 결정체
- 🎨 권장 톤: 전통·정통 톤. 노지 = 진짜 한라봉의 원형
- 📝 검증된 카피 샘플:
  - "노지에서 자란 한라봉, 진짜 맛이 다릅니다."
  - "한겨울 햇살을 그대로 머금은 깊은 단맛."
  - "5,000평 귤밭에서 직접 수확한 노지한라봉."
- 🌟 비전 연결: 겨울 매출 핵심. 단골 재구매 자산화 1순위.

### 🍊 하우스한라봉

- 💎 킬러 셀링 포인트: 후숙 불필요! 받자마자 바로 꿀맛 (노지와 핵심 차별점)
- 🎨 권장 톤: 자신감·차별화 톤. '기다림 없는 한라봉'
- 📝 검증된 카피 샘플:
  - "한라봉, 보통 며칠 후숙해야 단맛이 오르는데요. 저희 하우스한라봉은 다릅니다."
  - "후숙 필요 없이 받자마자 바로 꿀맛!"
  - "한라봉 후숙하다 무르게 한 적 있으시죠? 저희 하우스한라봉은 후숙 ZERO!"
- 🌟 비전 연결: 봄 매출 보강. 노지 시즌 끝나도 한라봉 단골 유지.

### 🌸 카라향 / 벚꽃카라향
- 📅 시즌: 3월~5월 (봄 제철)
- 💎 킬러 셀링 포인트: 당도·향 정점 / 새콤달콤 진한 풍미 / 5월 가정의 달 선물
- 🎨 권장 톤: 감성·선물 톤. 5월 가정의 달과 강력한 시너지
- 📝 검증된 카피 샘플:
  - "카라향, 지금이 1년 중 가장 맛있을 때!"
  - "어버이날·어린이날·스승의날 카라향으로 마음 전하세요."
  - ""이 귤은 어디서 났니?" 부모님이 먼저 물어보시는 카라향"
- 🌟 비전 연결: 5월 가정의 달 = 1년 중 선물 수요 최고 시즌. 재구매·신규 동시 폭발.

## Part 2. 옥수수

## Part 4. 향후 라인업 (27년까지)

### 🥩 고기류 (흑돼지·소고기·닭)
- 권장 톤: 제주 자연·청정 톤 강조
- 킬러 포인트: 제주 청정 환경에서 자란 / 친환경 사료
- 예시: '제주 흑돼지, 청정 자연이 키운 깊은 풍미'

### 🐟 생선류 (갈치·옥돔·방어·고등어·자리돔)
- 권장 톤: 신선·당일 출항 강조
- 킬러 포인트: 제주 앞바다 직송 / 새벽 경매 직후 발송
- 예시: '오늘 새벽 잡힌 제주 갈치, 그대로 보내드립니다.'

### 🏭 가공품 라인 (귤청·잼·드레싱·건귤·음료)
- 권장 톤: 정성·간직·선물 톤
- 킬러 포인트: 우리 농장 직접 가공 / 첨가물 최소
- 예시: '5,000평 귤밭에서 직접 키운 귤로 담근 귤청.'
- ⭐ 가공품은 해외 수출(4단계)의 핵심 라인업

## Part 5. 모든 상품 공통 차별화 포인트
모든 카피에 자연스럽게 녹여야 할 공통 자산:
- 5,000평 귤밭 직접 매입 추진 → '직접 키운'의 진정성
- 선과장 직접 운영 (2027) → '품질 직접 관리'
- 체험장 운영 (3,000명+ 방문) → '검증된 신뢰'
- AI 상담 + 사람의 따뜻함 → '친근한 CS'
- 귤나무 키우기 앱 → '재미있는 고객 경험'
- 취나물·애플초당부터 차근차근 → '검증된 품목만 판매'
- 자사몰 전용 혜택 → '찐 고객 우대'
- '아꼼이' 캐릭터/스토리 → '감성 차별화'
- 100% 무료 반품 보장 → '품질 자신감'
