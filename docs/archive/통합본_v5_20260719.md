## 운영 프로토콜 (Day 3 확립분 포함)
1. **복명복창**: 대표 요청 → 똑똑이 복창 → "ㄱ" → 등록 (사실 통보 예외)
2. **알림 읽는 법**: 도착(대기)·도착(진행)=무시 OK / **경과 갱신·도착(완료) = "ㄱ" 타이밍** / 알림 문구 복사+"ㄱ"도 OK
3. **이중 확인**: 클로드 코드 보고를 똑똑이가 관측 도구로 매번 교차 검산 (원본 데이터 대조 — Day 3에 시험지 결함·단가 현상을 이 루프로 적발)
4. **비용 발생 행동 = 승인 게이트**: 역량 실행·이미지·영상 생성 모두 "할까요? (약 X원)" → ㄱ → 실행 → audit
5. 지시 입력(마루)·역량 버튼 = 대표 전용 유지 (v5.1.1로 역량만 똑똑이 승인 게이트 실행 예정)
6. 조용할 때 응급: 폰 코드 탭(remote-control) 확인 → "지시함 폴링 재가동해줘" (단, Day 3 사례: 조용≠정지 — 긴 작업/외부 빌드 대기일 수 있음, 코드 탭으로 먼저 확인)
