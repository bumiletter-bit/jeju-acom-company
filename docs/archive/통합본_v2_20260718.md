# PART 1. 시스템 개요

- 회사 관리 프로그램(jeju-acom-company.onrender.com, Node/Express+PostgreSQL, Render 배포)에 통합된 AI 에이전트 관제 시스템. 사이드바 최상단, **대표 계정 전용** (프론트 숨김+백엔드 401/403)
- 화면: 픽셀 조직도 + 제목 옆 버전 칩[v5.0-dev] + 상시 지시 입력바("마루에게 지시하기") + LIVE 로그(결과 미리보기 클릭→모달) + 보고서함([✔확인]=soft-delete/다시보기) + 위젯(지식노트·피드백·오배정 주간·성적표)
- 비용: 개발=클로드 코드(Max 구독) / 운영=Anthropic API 크레딧 (~$17, 하루 풀가동 ~400원)
- 조직도(7/18 개편): 팀명 라벨이 팀장 **위**로. 대표 → [🏢기획팀] 마루(기획팀 실장) → [마케팅팀]한결→미소·글샘·예리 / [재무팀]한수→세미 / [법무팀]지율 / [개발부서]미래→기안
- 마루 생각 중 💭 말풍선(점 3개 애니메이션), 되묻기 🤔, 시험 중 📝 표시
