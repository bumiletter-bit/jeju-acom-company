- 일정 "(등록자)" 표기 = 정확한 동작, 수정 불필요 (대표 오해 확인 종결)
- 세미 숨은 상한 4곳 = 4.5단계 일괄 정리 (지금 안 걸리는 수준, 어차피 구조 손봄)
- 오퍼스 도입 보류: 소넷+판단표로 만점 달성, 필요성 소멸. (아이디어로 "고난도만 상위 모델 에스컬레이션" 구조는 기록해둠 — 당장 불필요)
- 지시함 릴레이: 자동 전달하되 복명복창 승인은 유지 (통로 자동화 ≠ 결재 자동화)
