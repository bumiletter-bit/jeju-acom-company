## 버전 계획

### v5.1 — 미소 원스톱 생성 (진행 중 — 지시 #26·#27)
- 생성 도구 = 제미나이 단일화 (힉스필드 보류 — 품질·기능 부족 확인 시에만 재검토)
- 대표 확정 모델 (중간급 통일): 이미지 기본 gemini-3.1-flash-image 1K(약 92원)/고급 gemini-3-pro-image(약 185원), 영상 기본 veo-3.1-fast 720p 8초(약 1,100원)/고급 veo-3.1 표준 1080p(약 4,400원)
- 건별 대표 승인 게이트 (승인 없이 생성 API 호출 불가 — 코드 레벨) → 생성 → 보고서함 📎(DB 보관·adminOnly) → 비용 audit
- 월 예상 (이미지 30 + 영상 4): 약 7,200원

### v5.1.1 — 똑똑이 역량 점검 실행 도구 (지시 #28-2)
- MCP 도구 `run_capability_test` — 역량 점검 실행 1가지만 (그 외 실행·쓰기 불가, 읽기 전용 3종과 분리)
- 통제 = 승인 게이트: 똑똑이는 대화에서 대표 승인("ㄱ") 후에만 호출 (운영 규칙 명문화) + 호출 전수 audit(mcp_run_test)
- ※ v5.1 태그 완료 후 착수 (순차 원칙)

### v5.2 — 한결 검수 게이트
- 글샘·미소 콘텐츠를 한결이 검수: ①지식 문서 규칙 준수(명분 없는 할인 금지 등) ②어그로(후킹) ③목적·전달사항 명확성 ④강조 포인트
- 검수 시에만 소넷 호출(건당 1회)로 비용 통제 → 통과 후 대표 승인 요청

### v5.3 — 예리 성과 분석
- 게시물 성과 데이터 입력 (코워크 조회 → AGENT OFFICE 입력, 반자동) → 잘된 것/안 된 것 비교 분석
- 미소·글샘용 전문 프롬프트 부여 → 교훈 후보는 기존 학습 승인제 경유 (대표 전용 · 요원당 활성 상한 10)

### v5.4+ — 대기 요원 전원 활성화
- 지율 (법무·노무): 대표 준비 노무사 지침서 투입
- 기안·개발부서 미래: 비전 자료 투입
- 각 활성화 착수 시 대표에게 자료 위치 확인 후 지식 문서로 투입
